# Doação Solidária – Sugeridor de Brinquedos

Projeto social desenvolvido por alunos do curso técnico em Desenvolvimento de Sistemas.

## Objetivo
Sugerir brinquedos de forma aleatória para crianças de 2 a 14 anos, considerando idade e sexo, auxiliando ações de doação em orfanatos.

## Tecnologias
- HTML
- CSS
- JavaScript

## Como executar
Basta abrir o arquivo `index.html` no navegador.
